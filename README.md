# Distance Calculator

Простой калькулятор расстояния на C# Windows Forms.

## Функциональность

- Ввод скорости (км/ч) через текстовое поле
- Выбор времени (часы) через TrackBar
- Расчёт расстояния по формуле: S = V × t
- Округление результата до двух знаков
- Автоматический пересчёт при изменении времени
- Обработка ошибок ввода скорости

## Интерфейс

- txtSpeed — поле ввода скорости
- trackBarTime — ползунок выбора времени
- lblDistance — отображение результата

## Пример работы

Скорость: 60 км/ч
Время: 2 часа
Результат: "Расстояние: 120.00 км"

## Куда можно встроить

- В навигационные приложения
- В транспортные калькуляторы
- В физические расчёты
- В логистические системы
- В обучающие проекты

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Расчёт происходит автоматически при перемещении ползунка времени. При некорректном вводе скорости выводится сообщение об ошибке.
